Usage

Step 1: Add Required Elements to the HEAD Element of Your Document

Include the jQuery and jqCoolGallery codebases as well as the jQuery $(document).ready() function where you'll invoke the plugin:

<head>
    
    <!-- include jquery -->
    <script type='text/javascript' src='http://ajax.googleapis.com/ajax/libs/jquery/1.7.1/jquery.min.js'></script>
    
    <!-- include jqcoolgallery -->
    <script type='text/javascript' src='jqcoolgallery/jquery.jqcoolgallery.1.8.js'></script>
    <link type="text/css" href="jqcoolgallery/jquery.jqcoolgallery.1.8.css" rel="stylesheet" />
    
    <!-- invoke the jqCoolGallery plugin inside $(document).ready() -->
    <script type="text/javascript">
    $(document).ready (function () {
        $('#gallery').jqCoolGallery ({}); // see below for the available options
    });
    </script>

    <!-- add whatever else you need in your document head -->
    
</head>

Step 2: Available Options

The following listing shows the available options for the plugin:

galleryTileWidthinteger: width of gallery tiles, default: 200
galleryTileHeightinteger: height of gallery tiles, default: 200
galleryColumnCountinteger: adds the "jqcg-gallery-item-last" class to the last item in each row - used to eliminate right margin, default: -1
galleryHoverCaptionboolean: whether or not to show/hide a gallery captions when hovering, default: true
galleryHoverExpandboolean: whether or not to expand gallery items when hovering, default: true
galleryHoverExpandPxinteger: number of pixels to expand gallery items (when galleryHoverExpand is true), default: 5
galleryTitleElementstring: HTML element used to display gallery title in panel, default: 'h3'
galleryDescElementstring: HTML element used to display gallery description in panel, default: 'p'
openGalleryinteger: 0 based index of gallery to open on initial view, default: -1
playSpeedinteger: milliseconds delay between slides when playing, default: 3000
animateInSpeedingeger: milliseconds to animate gallery tiles IN, default: 200
animateOutSpeedingeger: milliseconds to animate gallery tiles OUT, default: 200
initialFadeSpeedinteger: milliseconds to fade in gallery after loading, default: 2000
loopGalleriesboolean: true to auto-load next gallery after playing last slide in current gallery, default: false
imageAreaWidthinteger: width of the image area, default: -1
imageAreaHeightinteger: height of the image area, default: -1
imageAreaAlignstring: CSS alignment of image area - 'center' or 'left' or 'right', default: 'center'
imageAreaPaddingstring: CSS padding - '0' or '0 10px 0 0' or any other padding CSS to apply to image area, default: '0'
panelAreaWidthinteger: width of panel area, -1: don't apply width, default: -1
thumbLocationstring: 'left' or 'right' or 'top' or 'bottom', default: 'bottom'
thumbStylestring: 'list' or 'grid', default: 'list'
thumbAreaWidthinteger: only used for grid style, -1: use panelAreaWidth, default: -1
thumbHelperboolean: true to show a larger thumbnail (thumbnail helper) next to a thumbnail, default: true
thumbHelperMaxWidthinteger: max width of thumbnail helper, default: 100
thumbHelperMaxHeightinteger: max height of thumbnail helper, default: 100
thumbWidthinteger: width of a thumbnail container, default: 100
thumbHeightinteger: height of a thumbnail container, default: 100
thumbPaddinginteger: padding around thumbnail container, default: 0
thumbMargininteger: margin next to (and below-for thumbs in a grid) thumbnail, default: 4
thumbBorderWidthinteger: border width for thumbnail container, default: 1
thumbImageWidthinteger: used to specify width for all thumbnails, default: -1
thumbImageHeight// integer: used to specify height for all thumbnails, default: -1
thumbScrollButtonWidthdefault: 25,,
forceImageReloadboolean: true to reload all images, default: true
forceDemandReloadboolean: force reload of ajax content, default: true
forceDemandReloadParamstring: param used to force reload of ajax content, default: '_jqcg_'
htmlHomestring: HTML to use to bring the user back home from a slideshow, default: 'All Galleries'
homeToolTipstring: for gallery home text (see also htmlHome option), default: 'return to all galleries'
galleryTooltipstring: for gallery items, or use data-gallery-tooltip attribute in gallery item li element, default: ''
playToolTipstring: for the play button, default: 'start the slideshow'
pauseToolTipstring: for the pause button, default: 'pause the slideshow'
prevGalleryToolTipstring: for the previous gallery button, default: 'open the previous gallery'
nextGalleryToolTipstring: for next gallery button, default: 'open the next gallery'
nextSlideToolTipstring: for next slide button, default: 'next slide'
prevSlideToolTipstring: for previous slide button, default: 'previous slide'
rewindToolTipstring: for first slide button, default: 'first slide'
fnGalleryClickfunction: do your own thing when a gallery item is clicked. Return false to not open the gallery, default: null
keyboardInputboolean: listen for keyboard input, default: true
debugboolean: true to send debug info to the console, default: false

Here's an example of calling the plugin from $(document).ready():

$(document).ready (function () {
    $('#gallery').jqCoolGallery ({
        galleryTileWidth: 216,                            // width of gallery tiles
        galleryTileHeight: 150,                            // height of gallery tiles
        galleryColumnCount: 4,                            // eliminates right margin in last column
        galleryTooltip: 'Click To Open This Gallery',    // tooltip for gallery tiles
        imageAreaWidth: 675,                            // width of image area
        imageAreaHeight: 450,                            // height of image area
        imageAreaPadding: '0 15px 0 0',                    // padding between image area and panel
        panelAreaWidth: 240,                            // panel area width
        thumbWidth: 50,                                    // thumb width
        thumbHeight: 50,                                // thumb height
        thumbStyle:'list',                                // list style thumbnails
        thumbLocation:'top',                            // put thumbnails on top of slideshows
        debug: true                                        // sends debug messages to console
    });    
});

Step 3: Add the jqCoolGallery HTML Markup

Notes:

  • Put all the required jqCoolGallery markup inside its own <div> element.
  • It is highly recommended to use the width and height attributes for all images.
  • The prefered method for using jqCoolGallery for displaying a gallery of slideshows is to demand-load each slideshow.
  • Please see the demo pages for more examples.

Basic Setup - Demand Loaded

The first item should be an unordered list, which specifies each of the slideshows in the gallery. In this example, we will be demand-loading the images for each gallery using the rel attribute (see explanation below). Also, we are setting the CSS opacity for the list to be 0 to achieve the fade-in effect after the gallery loads.

<div id="gallery">
    <ul class="jqcg-gallery" style="opacity:0;">
        <li data-title="Gallery 1" data-desc="This Is An Amazing Slideshow!" rel="demo/galleries/gallery1.html">
            <img src="demo/galleries/gallery2/_thumb.jpg" alt="" />
        </li>
    </ul>
</div>

Demand Loading Content

The rel attribute specifies the URL that will supply the images as an HTML unordered list and will be pulled in using the jQuery.ajax() method. The following example shows the expected HTML markup that should be returned: (this is an unordered list with two images)

<ul>
    <li>
        <img src="demo/galleries/gallery1/5633.jpg" width="300" height="450" data-thumb="demo/galleries/gallery1/5633_thumb.jpg" alt="" />
        <div class="caption">This is the caption for the first image.</div>
    </li>
    <li>
        <img src="demo/galleries/gallery1/5656.jpg" width="300" height="450" data-thumb="demo/galleries/gallery1/5656_thumb.jpg" alt="" />
        <div class="caption">This is the caption for the second image.</div>
    </li>
</ul>
  • If the data-thumb attribute is not present, the full sized image will be used for the thumbnail instead. When possible you should use a smaller version of the image for a thumbnail because it will display better.
  • To specify the width/height of thumbnails, you can either include data-thumb-width and data-thumb-height attributes in the <img> tag, or specify thumbWidth and thumbHeight options when creating the plugin.
  • Captions are not required.

Gallery Tile Captions

Gallery tile captions are constructed from the data-title and data-desc attributes of the <li> element. However, you may format your own gallery tiles captions by adding a div with the caption class as follows:

<div id="gallery">
    <ul class="jqcg-gallery" style="opacity:0;">
        <li data-title="Gallery 1" data-desc="This Is An Amazing Slideshow!" rel="demo/galleries/gallery1.html">
            <img src="demo/galleries/gallery2/_thumb.jpg" alt="" />
            <div class="caption">
                <strong>Gallery 1</strong><br />
                You will love this one!
            </div>
        </li>
    </ul>
</div>
